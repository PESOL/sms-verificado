=============
SMS Verificados
=============

This module allow send SMS through SMS Verificados API.

Configuration
=============

To configure this module, you need to:

#. Go to setting and set SMS right to the users.
#. Now users with rights can see SMS Status menu.

.. figure:: path/to/local/image.png
   :alt: alternative description
   :width: 600 px

Usage
=====

To use this module, you need to:

#. Go to partners and use SMS composer wizard to send SMS. Then you can see in
SMS status menu all the sendend SMS.
